Консольная программа. Запускается с помощью команды "python main.py".

Если нет файла "in_U" (универсум), то программа предложит его создать.
Для задания универсума предусмотрен интерактивный интерфейс.

Далее если нет файлов "in_A" и/или "in_B", то программа предложит создать и заполнить их.
Для задания множеств A и B так же предусмотрен интерактивный интерфейс (отличный от того который использовался при задании универсума).
