Консольная программа. Запускается с помощью команды "python main.py".

Для начала работы необходимо иметь файл "in_U", где перечислены элементы
универсума (по одному на строке). (если файла нет, то программа выдаст ошибку).

Далее если нет файлов "in_A" и/или "in_B", то программа предложит создать и заполнить их.
